# E-Commerce Sales Trend Analysis and Predictive Modelling

## Overview
This project focuses on analysing e-commerce sales data and forecasting future revenue using machine learning techniques. The analysis helps identify sales patterns, business trends, and revenue performance across different product categories and regions.

## Project Objectives
- Analyse historical e-commerce sales data
- Identify sales and profit trends
- Visualise business performance using graphs and charts
- Build predictive models for revenue forecasting
- Support data-driven business decision making

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab
- Jupyter Notebook

## Dataset Features
The dataset contains the following columns:
- Order Date
- Product Name
- Category
- Region
- Quantity
- Sales
- Profit

## Project Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Sales Trend Analysis
- Profit Analysis
- Data Visualisation
- Predictive Modelling using Machine Learning
- Revenue Forecasting

## Machine Learning
Machine learning techniques were applied to analyse historical sales patterns and predict future revenue trends. Predictive modelling helps businesses improve planning, forecasting, and strategic decision-making.

## Outcome
The project demonstrates how machine learning and data analytics can help e-commerce businesses understand customer purchasing behaviour, monitor sales performance, and improve revenue forecasting accuracy.
